# Favicon Instructions

To add your custom favicon to the site:

1. Create or prepare a square PNG image for your favicon (recommended size: 192x192 pixels)
2. Name it `favicon.png`
3. Place it in the `/public` folder

The site is already configured to use this file as the primary favicon. The existing SVG favicon will serve as a fallback.

For optimal results, also consider adding:
- `apple-touch-icon.png` (180x180 pixels) for iOS devices

Note: These images should be placed directly in the `/public` folder to ensure they're available at the root path.
